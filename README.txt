心理年龄测试 - 使用说明

✅ 已完成功能：
- 24题心理测试（六大维度）
- 实际年龄输入与偏差计算
- 雷达图可视化
- 维度分析、优势、建议
- 微信群引流入口

🚀 如何测试：
1. 双击 index.html 打开
2. 点击"开始探索"
3. 输入生日
4. 完成24道题
5. 查看结果

📱 如何上线：
- 将整个文件夹上传到任何静态网站托管服务
- 推荐：GitHub Pages、阿里云OSS、腾讯云COS

💬 微信群配置：
- 准备一张群二维码图片
- 在 psychage-test 文件夹内创建 assets 文件夹
- 将二维码命名为 wechat-group-qrcode.jpg 放入 assets
- 如果不想用二维码，可以修改 result.html 中的微信号

🎨 自定义修改：
- 修改题库：编辑 data.js 中的 questions 数组
- 修改文案：编辑 result.html 中的内容
- 修改颜色：编辑 style.css
